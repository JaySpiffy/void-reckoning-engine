# Algorithmic Hierarchy Infrastructure

## Military Buildings

### Tier 1: Assembly Line
- **Cost:** 800 Requisition
- **Maintenance:** 20/turn
- **Build Time:** 3 turns
- **Capacity:** 5 Units
- **Unlocks:** [Cyber Skitarii], [Combat Servitor]
- **Effects:** 
  - +10% Production Efficiency
  - +1 Rank for Cyborgs
- **Prerequisites:** None
- **Lore:** Precision manufacturing floors where flesh is replaced with superior steel.

### Tier 2: Cyber-Foundry
- **Cost:** 2000 Requisition
- **Maintenance:** 50/turn
- **Build Time:** 5 turns
- **Capacity:** 5 Units
- **Unlocks:** [Walker Bot], [Grav-Platform]
- **Effects:**
  - +20% Vehicle production speed
  - +10% Unit Accuracy
- **Prerequisites:** [Tech_Algorithmic_Hierarchy_Industrial_Forging]
- **Lore:** Automated factories run by simple AI algorithms to construct war machines.
