# Unit Class Taxonomy - Void Reckoning

## Space Unit Classes

### Tier 1: Strike Craft
- **Fighter**: Dogfighting, high maneuverability
- **Bomber**: Payload delivery, anti-capital
- **Interceptor**: Fast response, bomber hunting

### Tier 2: Escorts
- **Corvette**: Anti-fighter screening
- **Frigate**: Electronic warfare, point defense
- **Destroyer**: Glass cannon, beam weapons

### Tier 3: Cruisers
- **Light Cruiser**: Patrol, screening
- **Heavy Cruiser**: Line combatant
- **Battlecruiser**: Fast capital hunter
- **Battleship**: Fleet anchor

### Tier 4: Capital Ships
- **Carrier**: Force projection
- **Iron-Sarcophagus**: Siege platform

### Tier 5: Super-Capitals
- **Titan**: God-engine of void
- **Planet Killer**: Strategic weapon
- **World Devastator**: Resource stripper
- **Stellar Accelerator**: Star manipulator

### Tier 6: Megastructures
- **Mothership**: Faction flagship

## Ground Unit Classes

### Tier 1: Infantry
- **Light Infantry**: Rapid deployment, objective control
- **Assault Infantry**: Close combat specialists
- **Skirmisher**: Hit-and-run, harassment

### Tier 2: Vehicles
- **Light Vehicle**: Recon, transport
- **APC**: Troop carrier
- **Anti-Tank**: Tank hunter

### Tier 3: Armor
- **Battle Tank**: Main battle line
- **Heavy Vehicle**: Mobile artillery
- **Superheavy Tank**: Breakthrough unit

### Tier 4: Command
- **Command Vehicle**: Buffs and coordination

### Tier 5: Walkers
- **Titan Walker**: God-engine of ground
- **Siege Engine**: City breaker

### Tier 6: Mobile Bases
- **Mobile Fortress**: Command center
